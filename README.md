# College Time Table Allocation System

A final-year Software Engineering project for automated college timetable generation, built with an MVC architecture and a clean separation between frontend and backend.

## Tech Stack

**Frontend:** HTML, CSS, Bootstrap 5, JavaScript
**Backend:** FastAPI (Python)
**Database:** MySQL + SQLAlchemy

## Folder Structure

- `backend/` — FastAPI application (API, Models, Services, Scheduler, Auth, Reports, Exports)
- `frontend/` — HTML/CSS/Bootstrap/JS client
- `database/` — Raw SQL schema, seed data, and constraints
- `docs/` — SRS, system design, and API documentation
